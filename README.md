## About The Project

![AIMusician Screenshot](https://firebasestorage.googleapis.com/v0/b/erin-mutchler-portfolio.appspot.com/o/Site%20Screenshot.PNG?alt=media&token=02877e97-8ca3-4cbf-90f6-45ccee8b273f)

AIMusician is currently in is beta. The functionality currently operating includes creating, deleting, and modifying songs. The AI only includes the bass playing AI and static drum samples. Future releases will include AI driven piano and drum accompaniment, improved AI musicianship (dynamics, articulation, etc.), selectable sample libraries for each instrument, and AI/human interactivity.
